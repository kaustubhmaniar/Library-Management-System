# 📚 Library Management System

A simple menu-driven Library Management System developed in C++ using Object-Oriented Programming (OOP) concepts.

## Features
- Add new book records
- Display all stored book records
- Input validation for Book ID, Price, and Pages
- Maximum capacity of 20 books
- User-friendly menu-driven interface

## Technologies Used
- C++
- Object-Oriented Programming (OOP)

## Project Structure

```
Library-Management-System
│── library.cpp
│── README.md
```

## How to Run

Compile:

```bash
g++ library.cpp -o library
```

Run:

```bash
.\library.exe
```
